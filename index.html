      • Installation

      • 1. Download the necessary files from the repo.
        2. Open Visual Studio Code.
        3. Open the terminal.
        4. Link the terminal to the main folder of the project.
        5. enter "npm install"
        6. enter "npm install inquirer"
        You are now ready to use the application.
